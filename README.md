Circuit-level simulation of a feed-forward neural network using LTSpice to model signal propagation delays and neuron behavior.

# Feed Forward Neural Network Simulation (LTSpice)

## Overview
This project models a simplified feed-forward neural network using analog circuit components in LTSpice. The objective was to replicate biological neuron behavior and study signal propagation delays using RC circuits.

## Features
- RC circuit-based neuron modeling
- 2:1 feed-forward neuron chain implementation
- Simulation of signal propagation delays
- Analysis of waveform outputs to study transmission characteristics

## Tech Stack
- LTSpice
- Analog Circuit Design
- Signal Simulation

## Results
The simulation successfully demonstrated time-delay characteristics similar to biological neural impulse transmission. Waveform analysis showed how signals attenuate and propagate through neuron stages.

## Files
- .asc files: LTSpice circuit designs
- Output waveforms and simulation plots
